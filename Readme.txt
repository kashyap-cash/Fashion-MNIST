To the run the code following are the requirements to be installed:
	1. Install Jupyter
	2. Install Tensorflow library
	3. Install keras library

To install tensorflow library following are the commands for mac OS:
	1. pip install --upgrade pip
	2. pip install tensorflow
	3. pip install tf -nightly

The code doesnot require to download the dataset as it directly downloads from the server.

Run the code and its all set!!!
